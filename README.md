# 🐍 Aprendizado - Curso Pycode

Este repositório foi criado com o objetivo de centralizar todos os meus estudos, exercícios, desafios e projetos desenvolvidos ao longo do curso **Pycode**. 

O foco principal aqui é registrar a minha evolução na linguagem Python, aplicando conceitos que vão desde os fundamentos até o desenvolvimento de soluções mais robustas.

---

## 🚀 Tecnologias e Ferramentas

* **Python 3.14.5** (Linguagem principal)
* **Git & GitHub** (Controle de versão e documentação)
* **Django** (Framework web)

---

## 📂 Estrutura do Repositório

O código está organizado da seguinte forma:

* `/fundamentos`: Conceitos básicos, estruturas condicionais, repetições e funções.
* `/exercicios`: Resoluções de desafios propostos durante as aulas.
* `/projetos`: Pequenas aplicações práticas construídas ao longo do curso.

---

## 🎯 Objetivos com este Repositório

1. **Prática:** Fixar o conteúdo teórico por meio de código puramente prático.
2. **Histórico:** Acompanhar a minha própria evolução técnica com o passar dos módulos.
3. **Portfólio:** Demonstrar minha dedicação e organização no aprendizado de novas tecnologias.
